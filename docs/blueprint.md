# **App Name**: EduEase AI

## Core Features:

- User Authentication: Secure login/signup for students and teachers with custom IDs and passwords. Students select batch and year; teachers select subject and batch.
- Admin Panel: Teachers' dashboard for adding/editing internal assessment marks, assignment submission dates, project work updates, and exam/practical dates.
- Workload Status: Display the current workload status for students based on deadlines and subject weightage. Use input provided by Teachers.
- AI-Powered Prioritization Tool: AI Tool to recommend which tasks (assignments, projects, etc.) students should prioritize based on deadlines, subject weightage, and professor-assessed difficulty level of subjects.
- AI-Powered Stress Reduction Suggestions: AI tool that analyzes project deadlines and calculates a stress score and can provide customized stress reduction suggestions (like moving project deadlines by small amounts), based on overall batch load.
- Batch Analytics: Dashboard for teachers with overall batch workload statistics and visualization tools
- Dark/Light Mode Toggle: Switch between bright and dark UI modes with specified color palettes. Theme toggle button will sit along side the 'sign in' and 'sign up' button.

## Style Guidelines:

- Light mode: Background is light green (#E0F2F7), Primary color is moderate green (#388364) and Accent is strong green (#68C8A2).
- Dark mode: Background is dark blue (#012C55), Primary color is moderate blue (#17507E), Accent is lighter blue (#3E7EB5).
- Font pairing: 'Apple Garamond' (serif) for the headlines and 'Cormorant Garamond' (serif) for body.
- Centered content for the body of the page and a responsive layout adaptable to different devices.
- Illuminate the sign-in button to draw focus.
- Subtle transitions and animations on UI elements.
- Use clear and consistent icons representing different academic tasks.
